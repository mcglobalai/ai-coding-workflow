# Living Claw

A protocol for canonically continuous digital individuals.

## 仓库结构

- `paper/` — 白皮书正文及编辑注释
- `log/` — 讨论记录（按日期，append-only）
- `decisions/` — 锁定的决策（按编号，immutable）
- `open/` — 未解决的问题（按 question）
- `inbox/` — 随手想法暂存

**→ [讨论进展快照](paper/living-claw.notes.md)** — Decided / Open / Not Yet Discussed
